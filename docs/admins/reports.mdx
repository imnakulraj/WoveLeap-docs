## Dashboard analytics

The main Dashboard provides a real-time overview:
- Total commissions this month / quarter / year
- Pending approvals count
- Top performing partners
- Sales pipeline by status

## Reports page

Go to **Reports** in the sidebar for detailed, exportable reports.

### Commission report
A full list of all commissions within a date range — filter by partner, sales person, status, or product. Export to Excel or CSV.

### Payout report
All payouts released within a period. Useful for reconciliation with your accounting system.

### Ledger
A running ledger of all commission entries — debits and credits — for audit trail purposes.

## Accounting exports

Go to **Reports → Accounting Export** to download data in formats compatible with:

| Software | Format |
|---|---|
| Tally | XML |
| QuickBooks | IIF |
| Zoho Books | CSV |

## Tax reports

Go to **Reports → Tax Reports** for:
- **TDS summary** — total TDS deducted per payee, per period
- **Form 26Q** — quarterly TDS return data (India)
- **Form 16A** — TDS certificates for individual partners (downloadable per partner)

## Analytics page

The **Analytics** page provides visual charts for:
- Commission trends over time
- Partner performance comparison
- Sales volume by product or region
- Monthly targets vs. actuals

Use the date range picker and filters to drill down into specific periods or segments.

title: Inviting Users

---

## Inviting a new user

1. Go to **User Management** in the sidebar
2. Click **Invite User**
3. Fill in:
   - **Full name**
   - **Username** — the login ID they will use (e.g. `john.smith`)
   - **Email address** — where the invite and notifications are sent
   - **Role** — Partner, Sales Person, or Admin
   - **Payout currency** — the currency in which this user will be paid
4. Click **Send Invite**

The user receives an email with their login credentials or a one-time link to set their password.

## Adding an SSO email

If you want a user to log in via Google or Microsoft SSO:

1. Open the user's profile in **User Management**
2. Find the **SSO Email** field
3. Enter their Google or Microsoft email address
4. Click **Save**

They can then use the Google or Microsoft button on the login page.

<Note>
  The SSO email must match the email address of their Google or Microsoft account exactly.
</Note>

## Managing existing users

From the User Management list you can:
- Edit a user's details or payout currency
- Reset their password
- Deactivate an account (they lose login access but data is retained)
- Delete an account (permanent — cannot be undone)

## Role permissions

You can customise what each role can see and do under **Settings → Role Permissions**. For example, you can restrict partners from seeing each other's data or allow sales persons to view commission breakdowns.
